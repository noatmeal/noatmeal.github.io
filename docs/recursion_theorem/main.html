A useful item to have in one’s toolbox while working in ZFC set theory111Short for Zermelo–Fraenkel set theory. See https://en.wikipedia.org/wiki/Zermelo-Fraenkel_set_theory for more information. is the ability to create inductive definitions. My goal in this post is to prove a theorem that allows us to do exactly that whenever the proper conditions are present. We’ll also work out a couple of fun examples at the end.

1.1 Natural Numbers

Definition 1.1.

A set I is inductive if and only if ∅∈I and ∀y∈I⁢(y∪{y}∈I).

Axiom 1.2 (Comprehension).

∀v∃y∀x(x∈y⇔x∈v∧ϕ(x)) where ϕ is a formula that y is not free in.

Definition 1.2.

Let I be a set who’s existence is guaranteed by Axiom 1.1. We use Axiom 1.2 to form the set

ℕ={y∈I:∀J⁢(J⁢ is inductive⇒y∈J)}

One should think of ℕ as the “smallest” inductive set222We will prove this in Theorem 1.3. An element of ℕ is called a natural number.

1.2 Peano Axioms

While we prove and refer to the following as theorems, it is common nomenclature to refer to these as The Five Peano Axioms.

Theorem 1.1.

1∈ℕ

Proof.

Since 1 is in every inductive set, then 1∈ℕ. ∎

Theorem 1.2.

If n∈ℕ, then n+∈ℕ.

Proof.

Let n∈ℕ. By definition ∀I⁢(I⁢ is inductive⇒n∈I), and so for any inductive set K we can conclude that n+∈K since n∈K. Recall that n∈I from Definition 1.2. By defintion, n+∈I. So we conclude from these two facts that n+∈ℕ. ∎

Theorem 1.3 (Principle of Mathematical Induction).

If S is an inductive subset of ℕ, then S=ℕ.

Proof.

We will argue by subset inclusions that S=ℕ. By hypothesis, we are given that S⊆ℕ. Now suppose that n∈ℕ. By hypothesis, S is an inductive set and so by definition of ℕ, n∈S. Hence, S=ℕ. ∎

Definition 1.4.

We refer to invoking Theorem 1.3 as an argument by induction.

Theorem 1.4.

For all n∈ℕ, we have that n+≠1.

Proof.

Let n∈ℕ. Recall that n+=n∪{n} and so n∈n+. Well n∉∅ and so it cannot be the case that n+=1. Therefore n+≠1. ∎

Definition 1.5.

A set T is transitive if and only if x∈T⇒x⊆T.

Lemma 1.1.

Every natural number is transitive.

Proof.

We argue by induction. Let S be the set of all transitive natural numbers. 1∈S trivially. Suppose n∈S. Since n+=n∪{n}, then for all x∈n+ either x∈n or x=n. If x∈n, then by our inductive hypothesis that n∈S, we know that x⊆n which allows us to conclude that x⊆n+. If x=n, then we also conclude that x⊆n+. Therefore, n+∈S which completes the inductive argument. ∎

Theorem 1.5.

If n,m∈ℕ and n+=m+, then n=m.

Proof.

Suppose the hypothesis is true. Well then n∈n+⇒n∈m+ and so either n∈m or n=m. By the same type of observation, either m∈n or m=n. If n≠m, then m∈n and n∈m. Well then Lemma 1.1 tells us that m⊆n and n⊆m and so n=m, a contradiction. Therefore it must be the case that n=m. ∎

2 Existence and Uniqueness

Theorem 2.1 (Recursion).

For any set X, if a∈X and f:X→X, then there exists a function u:ℕ→X such that u⁢(1)=a and u⁢(n+)=f⁢(u⁢(n)) for all other natural numbers n.

Proof.

Let 𝒞⊆𝒫⁢(ℕ×X) be defined such that (1,a)∈c and (n+,f⁢(x))∈c whenever (n,x)∈c for all c∈𝒞. It’s clear to see that 𝒞 is nonempty since ℕ×X∈C and so we can form the intersection of all sets in 𝒞 which we’ll call u.

Let S be the set of all natural numbers such that if n∈S then there exists (n,x)∈u and if (n,w),(n,y)∈u then w=y. We will prove inductively that S=ℕ which establishes that u is a function. Furthermore, given how u is constructed, such a proof gives us exactly the kind of function we are looking for.

Suppose that (1,b)∈u, a≠b, and consider the set M=u∖{(1,b)}. We claim that (n+,f⁢(x))∈M whenever (n,x)∈M. Since (1,a)∈M, this will give us that M∈𝒞 and u⊈M which is a contradiction to u being the intersection of all members of 𝒞. That will then allow us to say that (1,a)∈u and for all (1,b)∈u it would be the case that a=b.

So suppose that (n+,f⁢(x)) is not present in M for some (n,x)∈M. Well by the definition of M, for all α we have that (α∈u∧α≠(1,b))→α∈M which implies by contraposition that if α∉M, then α∉u∨α=(1,b). Well it can’t be the case that (n+,f⁢(x))=(1,b) by Theorem 1.4. So then we must conclude that (n+,f⁢(x))∉u which contradicts u’s construction since (n,x)∈u by the assumption that (n,x)∈M. We now must conclude that M∈𝒞 which gives us the base case for our inductive argument as discussed in the previous paragraph.

Now suppose that n∈S which implies that there is an (n,x)∈u for at most one x. It follows from the definition of u that (n+,f⁢(x))∈u. Now if n+ isn’t in S, then there exists (n+,y)∈u such that f⁢(x)≠y. Suppose that this is the case and consider the set J=u∖{(n+,y)} with some element (m,t)∈J.

If m=n, then t=x since n∈S and so (n+,f⁢(x))=(m+,f⁢(t))∈J. By construction of J, we note that α∉J→(α∉u∨α=(n+,y)) for all α. So if m≠n, then we know that m+≠n+ by Theorem 1.5 which means that if (m+,f⁢(t))∉J then we must conclude that (m+,f⁢(t))∉u which is a contradiction to the construction of u and so (m+,f⁢(t))∈J.

Therefore in every case, (m+,f⁢(t))∈J whenever (m,t)∈J. Since it also the case that (1,a)∈J since n+≠1 by Theorem 1.4, we conclude that J∈𝒞 and u⊈J which is a contradiction to u’s construction as being the intersection of all members of 𝒞.

So n+∈S and by the principal of mathematical induction we conclude that ℕ=S. ∎

Corollary 2.1.

The function defined in Theorem 2.1 is unique.

Proof.

Let a∈X and f:X→X for some set X. Suppose that there are two functions F:ℕ→𝕏 and G:ℕ→𝕏 given by Theorem 2.1
where F⁢(1)=G⁢(1)=a and for all other natural numbers n we have that F⁢(n+)=f⁢(F⁢(n)) and G⁢(n+)=f⁢(G⁢(n)). Let F⁢(1)=G⁢(1) be the base case for an inductive proof and suppose that F⁢(n)=G⁢(n) for some natural number n. Well then F⁢(n+)=f⁢(F⁢(n))=f⁢(G⁢(n))=G⁢(n+) by the inductive hypothesis. So by the principle of mathematical induction, F and G are the same function. ∎

3 Examples

Let’s define a couple of familiar concepts using what we’ve shown.

Definition 3.1.

For any natural number n∈ℕ, we have n+∈ℕ by Theorem 1.2. If we apply n+ and the successor function on ℕ to Theorem 2.1 and Corollary 2.1, we can define a unique function sn:ℕ→ℕ where sn⁢(1)=n+ and sn⁢(m+)=sn⁢(m)+ for all m∈ℕ.

Definition 3.2 (Addition).

Let +:ℕ×ℕ→ℕ be the function that maps +(n,m) to sn⁢(m) for all n,m∈ℕ. We almost always write this using inline notation as n+m and call it the sum of n and m.

Preliminary Remark:

In an effort to recover some familiar looking calculations we prove the following lemma and theorem.

Lemma 3.1.

n+1=n+

Proof.

n+1=sn⁢(1)=n+ ∎

Theorem 3.1.

n+(m+1)=(n+m)+1

Proof.
n+(m+1) =sn⁢(m+1)⁢ by Definition 3.2
=sn⁢(m+)⁢ by Lemma 3.1
=(sn⁢(m))+⁢ by Definition 3.1
=sn⁢(m)+1⁢ by Lemma 3.1
=(n+m)+1⁢ by Definition 3.2

∎

Example 3.1.

Recall Example 1.3. One can deduce that

  • •

    4=3+=3+1

Example 3.2.

Recall Example 3.1. We can see that

2+2=2+(1+1)=(2+1)+1=3+1=4
Definition 3.3.

For every n∈ℕ, we take sn:ℕ→ℕ from Definition 3.2 and define pn:ℕ→ℕ where pn⁢(1)=n and pn⁢(m+)=n+pn⁢(m).

Definition 3.4 (Multiplication).

Let ⋅:ℕ×ℕ→ℕ be the function that maps ⋅(n,m) to pn⁢(m) for all n,m∈ℕ. We almost always write this using inline notation as n⋅m and call it the product of n and m.

Preliminary Remark:

In an effort to recover some familiar looking calculations we prove the following lemma and theorem.

Lemma 3.2.

n⋅1=n

Proof.

n⋅1=pn⁢(1)=n ∎

Theorem 3.2.

n⋅(m+1)=n+(n⋅m)

Proof.
n⋅(m+1) =pn⁢(m+1)⁢ by Definition 3.4
=pn⁢(m+)⁢ by Lemma 3.1
=n+pn⁢(m)⁢ by Definition 3.3
=n+(n⋅m)⁢ by Definition 3.4

∎

Example 3.3.

Recall Example 3.1 once more. We can see that

2⋅3 =2⋅(2+1)
=2+(2⋅2)
=2+(2⋅(1+1))
=2+(2+(2⋅1))
=2+(2+2)
=2+4⁢ by Example 3.2
=2+(3+1)
=(2+3)+1
=(2+(2+1))+1
=((2+2)+1)+1
=(4+1)+1
=5+1=6
