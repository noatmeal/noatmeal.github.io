The Recursion Theorem

A useful item to have in one’s toolbox while working in ZFC set theory111Short for Zermelo–Fraenkel set theory. See https://en.wikipedia.org/wiki/Zermelo-Fraenkel_set_theory for more information. is the ability to create inductive definitions. My goal in this post is to prove a theorem that allows us to do exactly that whenever the proper conditions are present.

1 Preliminaries

Definition 1.1.

ℕ={1,2,…} and n+ is the image of n∈ℕ under the successor function222If this doesn’t make any sense, then a great review of this topic can be found in [1]. The proof of the main theorem for this post can be found in that text as well..

Axiom 1.1.

For all n∈ℕ, we have that n+≠1.

Axiom 1.2.

If n,m∈ℕ and if n+=m+, then n=m.

2 Statement and Theorem

Theorem 2.1.

For any set X, if a∈X and f:X→X, then there exists a function u:ℕ→X such that u⁢(1)=a and u⁢(n+)=f⁢(u⁢(n)) for all other natural numbers n.

Proof.

Let 𝒞⊆𝒫⁢(ℕ×X) be defined such that (1,a)∈c and (n+,f⁢(x))∈c whenever (n,x)∈c for all c∈𝒞. It’s clear to see that 𝒞 is nonempty since ℕ×X∈C and so we can form the intersection of all sets in 𝒞 which we’ll call u. Let S be the set of all natural numbers such that if n∈S then there exists (n,x)∈u and if (n,w),(n,y)∈u then w=y. We will prove inductively that S=ℕ which establishes u as a function. Furthermore, given how u is constructed, such a proof gives us exactly the kind of function we are looking for.

Suppose that (1,b)∈u, a≠b, and consider the set M=u∖{(1,b)}. We claim that (n+,f⁢(x))∈M whenever (n,x)∈M. If that weren’t the case then some (n+,f⁢(x)) would not be present in M for some (n,x)∈M. Well by the definition of M, for all α we have that (α∈u∧α≠(1,b))→α∈M which implies that if α∉M, then α∉u∨α=(1,b). Well it can’t be the case that (1,b)=(n+,f⁢(x)) by Axiom 1.1. So then we must conclude that (n+,f⁢(x))∉u which contradicts u’s initial construction since (n,x)∈u by the assumption that (n,x)∈M. Therefore, since (1,a)∈M and for any other (1,b)∈M we know that a=b, we conclude that 1∈S.

Now suppose that n∈S which implies that there is an (n,x)∈u for at most one x. It follows from the definition of u that (n+,f⁢(x))∈u. Now if n+ isn’t in S, then there exists (n+,y)∈u such that f⁢(x)≠y. Consider the set J=u∖{(n+,y)} and some (m,t)∈J. If m=n, then t=x since n∈S and so (n+,f⁢(x))=(m+,f⁢(t))∈J. By construction of J, we note that α∉J→(α∉u∨α=(n+,y)). So if m≠n, then we know that m+≠n+ by the Axiom 1.2 and so if (m+,f⁢(t))∉J then we must conclude that (m+,f⁢(t))∉u which is a contradiction to the construction of u. So n+∈S and by the principal of mathematical induction we conclude that ℕ=S. ∎

Corollary 2.1.

The function defined in Theorem 2.1 is unique.

Proof.

Let a∈X and f:X→X for some set X. Suppose that there are two functions F:ℕ→𝕏 and G:ℕ→𝕏 where F⁢(1)=G⁢(1)=a, F⁢(n+)=f⁢(F⁢(n)), and G⁢(n+)=f⁢(G⁢(n)) for all natural numbers n. Let F⁢(1)=G⁢(1) be the base case for an inductive proof and suppose F⁢(n)=G⁢(n) for some natural number n. Well then F⁢(n+)=f⁢(F⁢(n))=f⁢(G⁢(n))=G⁢(n+). So by the principle of mathematical induction, F and G are the same function. ∎
